# UI レビュー（2026-07-22 / #362 ATTRACT 導入の実ブラウザ検証）

#362（サイネージ→ATTRACT→受付開始の段階遷移）のマージ前検証として、iPad 縦
（810x1080, chromium, fake camera + fake mic）で **サイネージ→ATTRACT→受付/QR/
タイムアウト復帰/カメラ拒否の 8 インタラクション**を実ブラウザで自動検証し、
スクリーンショットを目視レビューした記録。検証は enroll 済み kiosk セッション +
サイネージ設定（clock + message）投入の実運用相当条件で実施。

## 検証結果（インタラクション）

8/8 PASS（マージ済みコードで再実行済み）:

1. サイネージ表示と presence-toggle の表示
2. fake camera の継続モーションで ATTRACT オーバーレイ表示（単発モーションでは非発火）
3. ATTRACT 中の背面サイネージ静止（2 秒間隔のスクリーンショットがピクセル一致）
4. `attract-start` タップ → オーバーレイ消灯 + 受付開始（用件選択へ）
5. ATTRACT 8 秒無操作 → サイネージ自動復帰
6. `attract-start-checkin` タップ → QR 受付へ切り替え
7. カメラ権限拒否時に ATTRACT が一切出ない
8. カメラ拒否時もタップで受付開始できる（タップ起動導線の維持）

## 検証で発見し **修正済み** の欠陥

### 🔴 F1. 待機サイネージが既定サイトで常に空（機能バグ）
`/api/kiosk/signage` と `SignageDisplay` のフォールバック siteId が `'default'` で、
seed サイト `'default-site'`（`lib/tenant/default-scope.ts`）と食い違い。KioskFlow の
クエリなし取得が常に空を返し、**サイネージ・presence 導線が既定構成で一切表示されない**。
#171 が潰した食い違いと同型の取り残し。→ default-scope へ統一 + route テストで固定。

### 🔴 F2. ATTRACT 挨拶文が背面サイネージ文言と重なり可読性崩壊
scrim（60% 黒）越しにサイネージの message 文言が透け、挨拶「こんにちは。受付をご利用
ですか？」と文字が重なる。→ 挨拶 + CTA を不透明パネル（`--color-surface`）に載せた。

### 🟠 F3. 待機フッターが「画面をタップして受付を開始」CTA と重なる
絶対配置フッター（QR 受付/退館/来訪検知）が iPad 縦で 2 段に折り返し、SignageDisplay の
受付開始 CTA を覆う。→ フッター実高さを ResizeObserver で計測し `bottomInsetPx` として
SignageDisplay に確保（escape バーと同一パターン）。

## 未対応の所見（軽微・次周回以降）

### 🟡 O1. SignageClock がキオスク選択言語に非連動
`toLocaleTimeString([])` がブラウザ/OS ロケール依存で、kiosk の言語切替（#327 で整備した
`locale`）に連動しない。実機は OS=ja 前提でほぼ顕在化しないが、#327 の残穴。
SignageItemView → SignageClock へ `locale` を配線すれば閉じる。

### 🟡 O2. 受付ステッパー第 4 步「確認」が「見やすさ設定」ピルに隠れる
iPad 縦の用件選択画面で、右上の見やすさ設定ボタンがステッパー第 4 ステップに重なる
（機能は無事、視覚のみ）。ステッパーの右余白 or ピルの配置調整で解消可能。

### 🟢 O3. ATTRACT パネル上端にサイネージ文言の端が僅かに覗く
パネル外は意図した scrim 表示であり実害なし。気になる場合は scrim を 60%→75% へ。

## 手順の再利用

検証スクリプトはセッション成果物（enroll → サイネージ投入 → 8 シナリオ）。実機 UAT（#65）
の合成カメラ代替手順としてそのまま流用できる。E2E 恒久化する場合は `tests/e2e/helpers.ts`
の `establishKioskSession` + fake-device 起動フラグ + `/api/admin/signage` PUT を組み合わせる
（getUserMedia モック基盤が無いため、検知発火は fake camera のローリング映像に依存する点に注意）。
